int uc16_to_utf8 ( int  c,
unsigned char *  utf8,
int  nbytes

Converts the given 16-bit unicode character c to utf8 and stores the result in the given utf8 buffer.

If more than nbytes are needed, the character is not converted and 0 is returned. Otherwise the number of bytes used in the buffer is returned. If the unicode character is not 16-bit, an error value -1 is returned.
