Detailed Description

A property view (id = 'prop') says that an object support get and set methods for properties.

Ay object supporting the 'prop' view can be used with get and put functions.

muse_cell(* muse_prop_view_t::put_prop) (muse_env *env, void *self, muse_cell key, muse_cell argv)

Modifies the value of the property identified by key to the given value.

The return value can be any value that the object deems relevant. It is generally the value itself. The return value should be documented in the specific object. Usually one doesn't need the return value, but occasionally it might be useful in an object specific way.
