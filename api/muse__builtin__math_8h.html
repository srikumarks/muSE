(rand M [N])

Generates a random number.

  • (rand N) returns an integer random number in the range [0,N)
  • (rand M N) returns an integer random number in the range [M,N)
  • (rand F) returns a float random number in the range [0,F).
  • (rand G F) returns a float random number in the range [G,F).
